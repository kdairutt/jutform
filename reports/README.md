# Investigation Reports

For each ticket you work on, you may optionally fill out an investigation report.

**How to use:**

1. Copy `TEMPLATE.md` to a new file named after the ticket (e.g., `TICKET-001.md`)
2. Fill in the sections as you investigate and fix the issue
3. Commit the report along with your code changes

These reports are **not required**. They help document your debugging process, especially if you ran out of time to fully fix an issue.
