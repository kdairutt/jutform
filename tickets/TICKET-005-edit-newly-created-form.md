# TICKET-005: Error when editing a form right after creating it

**Priority:** Medium
**Reported by:** Customer (username: `alice`, form: 168) via Support

> "I created a new form and clicked to edit it right away, and the page said something like 'not found.' I panicked and thought my form was gone, but when I refreshed a few seconds later it loaded fine. This has happened to me twice now."

Several users have mentioned this. It seems to only happen if you try to edit the form immediately — waiting a moment and then going back always works. Nobody has reported losing a form permanently.
